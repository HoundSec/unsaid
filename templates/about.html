About Unsent Project Clone

Welcome to the Unsend. This web application allows users to submit anonymous messages to people they have never sent messages to. Users can also search for messages by name and view them in a card format.

Features:

  • Submit anonymous messages with a name and message content.
  • Search for messages by name.
  • View messages in a card format with the name in bold and the message content below.

All messages are stored in a MongoDB database and are displayed in reverse chronological order.
